# Contributing to the AI-Native OT Security Framework

Thank you for your interest in contributing to this project. This framework is designed to be a community resource for securing American manufacturing and critical infrastructure, and contributions from the OT security, AI/ML, and manufacturing communities are essential to its success.

## How to Contribute

### Reporting Issues
- Use GitHub Issues to report bugs, suggest improvements, or ask questions
- For security vulnerabilities in the framework specifications, please email the maintainer directly (do not open a public issue)

### Specification Feedback
- The framework is currently in draft specification phase
- We welcome feedback on all specification documents via GitHub Issues or Pull Requests
- When suggesting changes to specifications, please include rationale grounded in real-world OT operational experience

### Contributing Code (Future)
- Reference implementations will be developed in later phases
- Code contributions will follow a standard PR review process
- All code must include tests and documentation
- Security-sensitive code requires additional review by a maintainer

### Contributing Test Datasets
- Benign OT traffic baseline datasets are especially welcome
- All contributed datasets must include provenance documentation
- Datasets must be anonymized (no real facility identifiers, IP addresses, or process data)
- Synthetic datasets generated from testbed environments (OpenPLC, GRFICSv2) are welcome

### Contributing Reference Architectures
- Reference architectures for additional manufacturing environments are planned
- If you operate in a manufacturing vertical not yet covered (process manufacturing, pharmaceutical, food & beverage, etc.), your domain expertise would be valuable
- Reference architectures should follow the template established in `reference-architectures/discrete-manufacturing.md`

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). By participating, you agree to uphold a welcoming, inclusive, and harassment-free environment.
